Cómo Crear un Proyecto en Django

Paso a Paso

Paso 1: Crear un Directorio para el Proyecto

Abre Visual Studio Code, abre una nueva terminal y navega al directorio donde quieres crear tu proyecto Django:

mkdir mi_proyecto_django
cd mi_proyecto_django

Paso 2: Crear un Entorno Virtual

Crea y activa un entorno virtual:

python -m venv venv
.\venv\Scripts\activate

Paso 3: Instalar Django

Instala Django en el entorno virtual:

pip install django

Paso 4: Crear el Proyecto Django

Crea un nuevo proyecto Django:

django-admin startproject mi_proyecto .

Paso 5: Ejecutar el Servidor de Desarrollo

Navega a la carpeta del proyecto y ejecuta el servidor de desarrollo:

cd mi_proyecto
python manage.py runserver

Paso 6: Crear un Repositorio Git (Opcional)

Inicializa un repositorio Git para tu proyecto:

git init
git add .
git commit -m "Initial commit"

Para ver el proyecto , deben ir a http://localhost:8000/
